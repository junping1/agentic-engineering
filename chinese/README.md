# Agentic Engineering

**构建生产级 AI Agent：Claude Code 架构解析**

本书通过拆解 [Claude Code](https://claude.ai/code) 在生产环境中验证的设计模式，总结出一套可落地的 AI Agent 构建指南，帮助开发者掌握构建复杂智能体系统的核心方法。

无论你在构建编程助手、研究型 Agent 还是自动化系统，都会面临相同的核心挑战：如何安全地赋予 Agent 工具能力？如何有效管理上下文？如何优雅地处理异常？本书通过经过大规模验证的架构模式来回答这些问题。

> **[English version](../README.md)**

## 目标读者

熟悉 API 和程序控制流，但尚未接触过 AI Agent 开发的软件工程师。无需 Agent 开发经验，只需具备基本编程知识和对大语言模型的初步了解。

## 阅读方式

你可以按顺序阅读以建立完整的认知体系，也可以在遇到具体设计问题时直接跳转到相关章节。每章先介绍概念，再展示 Claude Code 的实现方式，最后提炼出可复用的通用原则。
